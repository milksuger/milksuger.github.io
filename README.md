# Cursor提示词网站

这是一个展示Cursor提示词的响应式网站，使用原生HTML、CSS和JavaScript构建。

## 项目结构

```
.
├── index.html          # 主页面
├── styles/            # 样式文件目录
│   └── main.css      # 主样式文件
├── js/               # JavaScript文件目录
│   └── main.js      # 主JavaScript文件
└── README.md        # 项目说明文件
```

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)

## 特性

- 响应式设计
- 语义化HTML
- 可访问性支持
- 现代化UI/UX
- 性能优化

## 开发指南

1. 克隆仓库
2. 使用现代浏览器打开 index.html
3. 开始开发！

## 浏览器支持

- Chrome (最新版)
- Firefox (最新版)
- Safari (最新版)
- Edge (最新版)
